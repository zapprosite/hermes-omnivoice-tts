# Hermes Omnivoice Tts

**Engine TTS (OmniVoice client + sub-sentence streaming) para hermes-agent**

## Install

```bash
pip install hermes-omnivoice-tts
hermes_omnivoice_tts_bootstrap  # se disponível
```

## Compatibilidade

- Python 3.11+
- hermes-agent >= 0.16.0
- Ubuntu 22.04+
